## [1.0.0]

### Added

- Project conception!
- Rendering and system event handling implementation via Simple DirectMedia
  Layer library.
- UI implementation via Dear ImGUI library.
- Program settings saved in user's home directory.
- Input handling from keyboard via SDL's Keyboard API.
- Input handling from game controllers via SDL's Joystick API.
- Build recipe for Linux and for MS Windows via CMake.
- Configurable video modes, resolutions and screen displays.
- In-program logging of joystick events.
- Dynamic handling of disconnecting and reconnecting game controllers.
- Capped or uncapped FPS.
